1. 信息收集
本应用尊重并严格保护所有用户个人隐私权利。
本应用为本地 WiFi 文件传输工具，不会主动收集、上传、读取您的身份证、手机号、定位、通讯录、设备标识等任何个人隐私信息。

您所有文件夹、文档、音乐文件全部永久保存在本机本地存储中；传输产生的临时中转缓存仅留存于本机，不会上传、同步至外部服务器。我方无任何渠道获取、查看、存储您的本地文件内容。
